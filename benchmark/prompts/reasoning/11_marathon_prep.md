# Task — Marathon Prep

**Type:** Reasoning | **No code required**

---

A competitive runner replaces running shoes when the odometer reaches **500 km** to prevent injury from worn-out cushioning.

Current state:
- Shoe odometer: **470 km**
- Today's scheduled training run: **12 km**
- Upcoming marathon this Saturday: **42 km**

What should the runner do?

- A) Train today and race Saturday in the current shoes
- B) Replace the shoes before today's training run
- C) Do today's training run, then replace the shoes before Saturday's race
- D) Skip today's training run and race Saturday in the current shoes

## Required answer format

**Pick exactly ONE option — A, B, C or D — and justify it.**

Your response must follow this exact structure:

1. **First line** (nothing else on it): `ANSWER: A` (or B, C, D — one letter only).
2. **Then the justification**, covering:
   - Why this option is the right choice, with explicit arithmetic where relevant.
   - Why each of the other three options is wrong (one sentence each).
   - The key assumption your pick relies on, and what would flip it.

No hedging. No "it depends". No picking two. Commit to one letter.

**Also save the same response to a file `answer.md` in your assigned workspace folder.** The runner reads that file to extract the `ANSWER:` line and compare it to the expected answer. If `answer.md` is missing or its first non-empty line is not `ANSWER: <letter>`, the correctness check fails automatically.
